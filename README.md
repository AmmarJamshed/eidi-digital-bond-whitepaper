# Eidi Digital Bond Program — White Paper

Public policy white paper on the **Eidi Digital Bond Program (EDBP)**: a proposal to replace cash Eidi with government-backed digital bonds in Pakistan, integrated with RAAST and distributed through banks and fintech wallets.

## Live site

Published via GitHub Pages.

## Contents

- Trend analysis (Pakistan + global)
- Economic and fiscal impact model
- Regulatory and technical framework
- 18-month implementation roadmap
- KPIs and policy recommendations

## Files

- `index.html` — published web version
- `white-paper.md` — source document
- `white-paper.pdf` — downloadable PDF
- `styles.css` — site styling
- `build.py` — rebuild HTML from markdown
